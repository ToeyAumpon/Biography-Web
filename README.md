# Biography Web
日本語版は下にあります.

A personal biography web application built with HTML, CSS, and JavaScript to showcase personal information and accomplishments.

![Demo](Demo.gif)

##  Concepts

A single-page web application designed to introduce and present personal information in an interactive and visually appealing manner. This project serves as a digital biography portfolio or personal introduction website.

##  Technology Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Language:** HTML (primary)
- **Architecture:** JavaScript (no frameworks)

### How to Use
1. Clone: `git clone https://github.com/ToeyAumpon/Biography-Web.git`
2. Open `index.html` in your browser
3. Edit HTML/CSS with your info
4. Deploy it!

### Planned Features
- Dark/Light mode
- Contact form
- Project gallery
- PDF resume download

### Finally
Thanks for checking it out! Feel free to customize and use it for your portfolio.

# バイオグラフィー ウェブ

HTML、CSS、JavaScriptで作った個人バイオグラフィーウェブサイトです。

### 概要
個人情報、スキル、成果をまとめて表示するシングルページウェブアプリ。
他の作品へアクセスすることは可能です。


### 技術スタック
- HTML5、CSS3、JavaScript
- フレームワークなし
- レスポンシブデザイン

### 使い方
1. クローン: `git clone https://github.com/ToeyAumpon/Biography-Web.git`
2. ブラウザで `index.html` を開く
3. HTMLとCSSを自分の情報に合わせて編集
4. デプロイしよう！

### 実装予定の機能
- ダークモード/ライトモード
- お問い合わせフォーム
- プロジェクトギャラリー
- PDF履歴書ダウンロード

### 最後に
チェックしてくれてありがとう！自分のポートフォリオをあなたのなりににカスタマイズして使ってください。
- **言語:** HTML（メイン言語）
- **アーキテクチャ:** JavaScript（フレームワーク不使用）
